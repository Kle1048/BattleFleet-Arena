Lade GLBs …

Die Hülle projiziert alle Mesh-Vertices der skalierten GLB-Gruppe auf XZ (Welt), wie nach fitIslandToGameplayRadius im Spiel. Hohe Geometrie (Bäume, Türme) verbreitert die Hülle in XZ — bei Bedarf Eckpunkte im JSON manuell straffen. Ziel im Repo: shared/src/data/mapIslandPolygonOverrides.json — Array islands mit id (i1…i5) und verts mergen; leeres Array = nur prozedurale Polygone. Der Browser schreibt nicht direkt ins Projekt: Download oder Zwischenablage, dann Datei im Editor ersetzen/mergen.
